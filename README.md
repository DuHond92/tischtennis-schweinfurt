Vision

PlattenTreff soll die erste Anlaufstelle für Tischtennisspieler werden, die spontan spielen, neue Menschen kennenlernen und hochwertige Tischtennisplatten entdecken möchten.

Jede Funktion der App soll dieses Ziel unterstützen.
