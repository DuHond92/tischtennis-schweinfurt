Tags & Chips

Tags benennen wenige wichtige Merkmale. Chips verändern Filter und zeigen ihren Auswahlzustand.

Warum

Schnelles Erfassen

Tags machen Spielart, Inhaltstyp oder Status scanbar, ohne die Hauptinformation zu verdrängen.

Wann nicht

Keine Metadaten-Sammlung

Normale Informationen wie Ort und Uhrzeit bleiben Text. Nicht jedes Datenfeld wird zu einem Tag.

Nur das Wichtigste zeigen

× Vermeiden

Hierarchie zerstören

  • Nicht sechs Tags gleichzeitig zeigen
  • Keine Bedeutung nur durch Farbe
  • Tags nicht als Buttons tarnen

Barrierefreiheit

Interaktive Chips sind echte Buttons und verwenden aria-pressed. Entfernen-Aktionen benennen den betroffenen Filter.

Codebeispiel

HTML anzeigen
<button class="efc-chip efc-chip--active" aria-pressed="true">Punktspiel</button>
