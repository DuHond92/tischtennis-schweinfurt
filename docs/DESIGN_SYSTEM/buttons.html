Buttons

Buttons lösen eine unmittelbare Aktion aus und zeigen deren Priorität eindeutig.

Warum

Eine Entscheidung ermöglichen

Ein Button beschreibt mit einem Verb, was nach dem Auslösen passiert. Die visuelle Variante entspricht der Bedeutung.

Wann nicht

Keine Navigation vortäuschen

Führt ein Element zu einer anderen Seite, wird ein Link verwendet. Tags und Cards ersetzen keine Buttons.

Barrierefreiheit

Icon-Buttons benötigen einen zugänglichen Namen. Ladezustände verwenden aria-busy; deaktivierte Aktionen bleiben verständlich begründet.
